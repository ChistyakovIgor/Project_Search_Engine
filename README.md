# Search Engine version 1.0
The file information search system for a local network

## The opportunities
- the keyword search
- the processing user requests

## The stack of necessary requirements:
- C++ / nlohmannjson / googletest / CMake
- Clion
- complier Clang

## The installation
- Clone a repository:
  git clone https://github.com/ChistyakovIgor/Project_Search_Engine.git
- Go to the folder:
  cd project_search_engine
